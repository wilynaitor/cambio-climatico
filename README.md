# cambio-climatico
trabajo final python pro

## Esta es la idea de mi proyecto:
> Problema: La falta de conciencia ambiental de las personas es un problema grave,
> muchas personas se quejan del cambio climatico pero no se molestan en reciclar
> cosas que siempre estan en la casa.
>
> Solucion: Motivar a las personas a reciclar papel y plastico,
> incentivando y haciendo visible el esfuerzo de otras personas
>
> ## Este es mi tipo de proyecto:
